YouTube™ Player Extension
버전 1.0.0
YouTube의 플레이어 기능을 향상시키고
실용적인 오디오 제어 기능을 제공하는 확장 프로그램
🎵 주요 기능
  • 오디오 컴프레서 - 볼륨 증폭 및 다이나믹 레인지 압축 (50% ~ 200%)
  • 스테레오 패닝 - 좌우 음향 균형 조정 (L/R)
  • Picture-in-Picture - 작은 창으로 비디오 시청
  • 미니플레이어 - 스크롤 시 작은 플레이어로 계속 시청
  • 자동 품질 선택 - 선호 화질로 자동 설정
  • 비디오 코덱 선택 - H.264, VP9, AV1 중 선택
💡 개발 배경

기존의 복잡한 YouTube 확장 프로그램들과 달리, 정말 필요한 기능만 엄선하여 성능과 안정성에 중점을 둔 설계입니다.

🎤 노래방 기능이 필요하다면?
Transpose ▲▼ 추천
노래 키 조정, 템포 변경, 반복 구간 설정 등의 노래방 기능이 필요하시면 Transpose ▲▼ 확장 프로그램을 추천드립니다.

각 확장 프로그램이 각자의 전문 분야에서 최고의 성능을 발휘할 수 있도록 역할을 분담했습니다.

🛠️ 기술 정보
  • Manifest V3 - 최신 크롬 확장 프로그램 표준
  • 모듈화 설계 - 기능별 분리된 JavaScript 모듈
  • Web Audio API - 고품질 오디오 처리
  • 메모리 최적화 - 사용하지 않는 기능은 로드하지 않음
  • YouTube 호환성 - YouTube 업데이트에 대응하는 안정적인 구조
